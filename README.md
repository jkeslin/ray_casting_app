# Point-in-Polygon Checker

## Overview

The **Point-in-Polygon Checker** is a simple web application that determines whether a given point (specified by x and y coordinates) is inside or outside a predefined polygon. It utilizes the Ray Casting Algorithm for its calculations.

## Features

- Input x and y coordinates for a point.
- Checks if the point is inside or outside a given polygon.
- Real-time results displayed upon form submission.
